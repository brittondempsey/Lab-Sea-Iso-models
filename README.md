This directory contains the Matlab script used to simulate surface nitrate drawdown and associated stable isotopic trends over the progression of the 2022 Labrador Sea spring bloom.

Author: Britton Dempsey

Email: brittondempsey@dal.ca

This code was written to support the findings of the paper "Nitrate stable isotopes complement subarctic new production estimates" published in Nature Communications Earth and Environment (DOI: 10.1038/s43247-026-03353-x). The main script is found within the "BELAS_1 New production model" folder, and is titled "CE22_NO3up_d15N_d18O_newproduction_model_working_260211.m". The optimized parameters used to produce model outputs discussed in the associated manuscript are listed in the .csv files. 

The data supporting the findings of this study are available in the PANGAEA data repository at https://doi.org/10.1594/PANGAEA.986850.
